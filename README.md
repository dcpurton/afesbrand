# afesbrand

The `afesbrand` LaTeX package provides a set of macros for producing
components of the AFES Brand guidelines.

## Installation

`afesbrand` uses the `l3build` system.

Clone the git repository using:

```
git clone https://github.com/dcpurton/afesbrand.git
```

Change to the `afesbrand` directory, and then the style file
(`afesbrand.sty`), its support files and documentation (`afesbrand.pdf`) can
be installed by running:

```
l3build install --full
```
